# AI Agent Workflows (LLM + Automation)

Production-style AI Agent workflow experiments combining LLMs with automation tools.

---

## 🚀 Overview

This repository demonstrates practical AI-powered automation pipelines built with LLMs and structured tools:

* PoE Build Analyzer (Game Data)
* Script Translation Agent (.rpy processing)
* Probability Simulation Tools
* General Agent Workflow Design

---

## 🧠 Architecture

Core pipeline:

Task → Decomposition → Tool Use → LLM Reasoning → Output Validation

### Components

* **LLM Layer**: reasoning, generation, structured output
* **Tool Layer**: parsing, simulation, structured processing
* **Agent Layer**: orchestration and workflow control

---

## 🛠 Features

### 1. Build Analyzer

* Parse structured input data
* Analyze stats and modifiers
* Generate optimization suggestions using LLM

### 2. Translation Agent

* Extract structured text patterns
* Translate with context awareness
* Preserve original file structure

### 3. Simulation Tools

* Probability estimation
* Expected cost calculation
* Strategy suggestions

## ⚙️ Setup

1. Clone the repository:

```bash
git clone https://github.com/yourname/ai-agent-workflows.git
cd ai-agent-workflows
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Configure environment variables:

Create a `.env` file in the root directory:

```bash
OPENAI_API_KEY=your_api_key
```

---

## ▶️ Run

```bash
python main.py
```
